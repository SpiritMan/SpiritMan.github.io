Chrome插件推荐

Momentum

提供了漂亮的壁纸，每天换一图，还提供了简单的TODO List，天气等功能

Dribbble New Tab

非常受设计师的喜欢，因为每次一打开新标签可以让你打开空白tab的时间第一时间把每日精选作品展现出来，视觉的享受

PostMan

一款功能强大的API开发调试的工具。不论你是做后端开发还是客户端开发，PostMan可以完成所有的HTTP请求，支持集成测试，最重要的是数据可以云同步，以及生成API接口文Demo 。

JSONView

JSONView 是一个方便查看 Json 结构的插件，展开，折叠，看接口返回数据非常方便 Demo

Octotree

当我们在浏览别人的开源代码时，还要clone下来一个文件查看，而有了这个插件，你可以直接在Chrome侧边栏向打开文件夹一样的查看别人的项目

印象笔记·剪藏

一键保存网页到印象笔记，即时同步到你的手机和电脑上，不用复制黏贴再整理

惠惠购物助手

在您网购浏览商品的同时，自动对比其他优质电商同款商品价格，并提供商品价格历史，帮您轻松抄底，聪明网购不吃亏

The Great Suspender

he-Great-Suspender 给我们这些喜欢开n个页面一直不关的人带来了福音，它可以自动挂起那些没有使用的Tab页面，并把内存释放出来，当你重新加载这些页面的时候才分配内存。这样下来，哪怕我开几十个页面都不觉得卡了

AdBlock AdBlock Plus

用来屏蔽网页上的广告和视频中的广告

Vimium

鼠标就像腹部的赘肉一样既不健康容易养成鼠标手使用起来又低效。如果你是Vim 爱好者，那么你一定会爱上Vimium 的，用更加Hack的方式操作网页，配上机械键盘，五指如飞，游览器所有的功能都可以通过快捷键来完成Thor

Thor 是一款提供了全局快捷键设置的小工具，直接点击左下角的添加按钮选择软件，再录制快捷键，这样设置就算是完毕了。当你需要启动某个软件时，只需要按下之前设置的快捷键就可以快速打开。
